# HarelExam Automation Project - Clarifications

## 1. Running the Tests
The test script can be run either from an IDE(in my case Intellij), or by using the command:
```bash
mvn test
```
Test results can be views with Allure using:
```bash
allure serve allure-results
```

## 2. Spring Boot Usage
This project is built using Spring Boot(Spring initializer).
